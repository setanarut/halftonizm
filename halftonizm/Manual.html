Halftonizm

Generates flow map-based wave/halftone animations, provides previews, and exports as GIF/APNG/PNG.

Quick Start

  1. Flow map Layer: Select the layer that determines the flow.
  2. Artwork layers: Select the group layer for color blending.
  3. Adjust parameters, then generate using Build Frames.
  4. Watch with Play, and save via Save GIF/APNG/Image Seq.

Parameters

  • Preset: Selects a predefined setting.
  • Wave count: Wave density. Higher values result in denser lines.
  • Total frames: Total number of animation frames.
  • Waveform: Wave type (triangle, sawtooth, sine).
  • Reverse: Reverses the animation direction.
  • FPS: Playback and recording frame rate.
  • Blending mode: Blending type (HardMix or Smoothstep).
  • HardMix: Toggles artwork layer blending on/off.
  • Flow map Layer: Source layer used to read the wave phase.
  • Artwork layers: Group for base + top layer blending.
  • Scale: Output resolution percentage.
  • Output size: Final resolution based on the scale.

Buttons

  • Build Frames: Generates all frames.
  • Preview First Frame: Generates only the first frame.
  • Play: Play/stop the preview.
  • Save GIF: Saves as a GIF file.
  • Save APNG: Saves as an APNG file.
  • Save Image Seq: Saves frames as a PNG sequence.
  • Install NumPy: Installs NumPy (for faster calculations).
  • Install Pillow: Installs Pillow (for bicubic scaling and fast export).
